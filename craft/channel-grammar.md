# Channel grammar — universal reference

**Read by**: Creative Director (Concept §6 channel rollout), all 5 Copywriters, Social/Community Manager, future Static Designer + Motion Designer, future Web / Press / Influencer specialists.

Each channel has its own physical and cognitive constraints. The channel-grammar reference captures the constants that the *creative* must work within — aspect ratio, glance time, sound default, motion default, dwell time, viewing context. Get these wrong and even great work fails the surface.

## How to use this reference

Each channel entry has the same fields:

- **Aspect ratio**: primary + acceptable secondary ratios
- **Glance time**: how long the viewer typically engages before deciding to keep / skip
- **Dwell time**: if engaged, how long the viewer's attention is available
- **Sound default**: on / off / variable — assume which one when designing
- **Motion default**: static / motion / variable
- **Viewing context**: where + how the viewer is consuming (mobile / desktop / billboard / TV / cinema)
- **Word-count ceiling**: rough copy ceiling that fits the surface
- **Typical use cases**: what this channel is genuinely good for
- **Pitfalls**: what fails on this surface even when the work is good

## Out-of-home (OOH)

Billboards (static + digital), transit (bus / metro / airport), wild posting, street furniture.

| Field | Value |
|---|---|
| **Aspect ratio** | Varies wildly: 14:48 (US bulletin), 6-sheet (UK portrait), 48-sheet (UK landscape), 9:16 (digital portrait), 16:9 (digital landscape) |
| **Glance time** | 1.5–3 seconds (highway billboard); 5–15 seconds (transit dwell); 30s+ (rail platform repeat-view) |
| **Dwell time** | Same as glance — single exposure, no scroll-back |
| **Sound default** | None (silent surface) — exception: digital OOH with audio is rare and contextually inappropriate |
| **Motion default** | Static (traditional); motion + transitions on digital OOH (sequence of 3–8 static frames common) |
| **Viewing context** | In motion (driver / commuter / pedestrian); ambient (waiting room / station) |
| **Word-count ceiling** | 6–8 words for highway billboards; 12–20 for transit at-eye-level |
| **Typical use cases** | Brand awareness; campaign launches; product launches with simple proposition; political / cultural messaging |
| **Pitfalls** | Too much copy — if you can't read it in 2 sec at 60mph, it fails. URLs that aren't memorable. QR codes assumed to be scanned (rare in motion). Wit that requires re-reading. |

## Broadcast video

TV (linear), connected TV (Netflix / Hulu / Roku ads), pre-roll on streaming.

| Field | Value |
|---|---|
| **Aspect ratio** | 16:9 (always) |
| **Glance time** | 0–3 seconds (skip moment for skippable; ad-break re-attention for non-skippable) |
| **Dwell time** | 15 / 30 / 60 / 90 / 120s standard lengths |
| **Sound default** | ON — broadcast/CTV viewers expect audio |
| **Motion default** | Motion always |
| **Viewing context** | Lean-back (sofa / shared screen); often with second-screen device alongside |
| **Word-count ceiling** | Voice-over: ~150 wpm spoken; 30s spot = ~70 words VO; supers (on-screen text) brief and few |
| **Typical use cases** | Brand-building; mass-reach; emotional storytelling; product launches with cinematic ambition |
| **Pitfalls** | Skippable formats kill un-front-loaded spots — hook in first 3 sec or wasted spend. Heavy second-screen distraction. Production cost ceiling forces creative concessions. |

## Cinema

In-theatre pre-feature advertising.

| Field | Value |
|---|---|
| **Aspect ratio** | 1.85:1 or 2.39:1 (theatrical; not 16:9) |
| **Glance time** | n/a — captive audience, no skip |
| **Dwell time** | Full ad runtime (30s / 60s / 90s common) |
| **Sound default** | ON — full Dolby sound system |
| **Motion default** | Motion always; cinema cinematography aesthetic expected |
| **Viewing context** | Dark room, large screen, captive audience, audio enveloping |
| **Word-count ceiling** | Same as broadcast — VO at ~150 wpm |
| **Typical use cases** | Premium brand storytelling; entertainment-adjacent products; mass-reach with emotional craft; matched to film context (action movies / dramas / family) |
| **Pitfalls** | Cost-per-impression high. Wasted if creative is broadcast-recycled — cinema rewards craft that broadcast TV doesn't. Audience often half-watching as they settle in. |

## Print

Magazine, newspaper, branded longform inserts.

| Field | Value |
|---|---|
| **Aspect ratio** | Magazine: full-page bleed (A4 / US Letter); double-page spread (2:1ish); half-page / quarter-page partials. Newspaper: broadsheet / tabloid / column-units |
| **Glance time** | 1–3 seconds (page-turn) |
| **Dwell time** | If hooked: 30–90 seconds for an ad; 3–8 minutes for a feature-length branded piece |
| **Sound default** | None |
| **Motion default** | Static (paper); some print runs with foil / die-cut for tactile effect |
| **Viewing context** | Lean-back, lean-forward (commute / coffee shop / waiting room) |
| **Word-count ceiling** | Headline ≤8 words; body 30–150 words for an ad; 500–3,000 for branded longform |
| **Typical use cases** | Brand reinforcement in trusted titles; consideration-stage proof (a longform B2B case study in The Economist); luxury / craft brands; political |
| **Pitfalls** | Declining readership unless target audience is print-loyal. Lead times (4–8 weeks). Cannot A/B-test. Recall lower than digital for most product categories. |

## Direct mail

Postcards, letters, packages.

| Field | Value |
|---|---|
| **Aspect ratio** | Postcard (4×6, 5×7, 6×9); letter (US Letter / A4 folded); package (3D, varied) |
| **Glance time** | 2–5 seconds before sort-to-recycle |
| **Dwell time** | If kept: 30s–3 min; if filed, days |
| **Sound default** | None |
| **Motion default** | Static; novelty 3D / scratch-off / textured can boost dwell |
| **Viewing context** | Mailbox sort; lean-back at home |
| **Word-count ceiling** | Postcard: 20–40 words; letter: 200–500; package: handled like product packaging |
| **Typical use cases** | High-value direct response (real estate, B2B account-based marketing, premium consumer); event invites; political ground game |
| **Pitfalls** | Cost-per-piece high. Print + mail lead time. Open-rate low in households trained to discard. Compliance (CAN-SPAM doesn't cover mail, but state laws exist). |

## Packaging

Product packaging, label, box, wrapper.

| Field | Value |
|---|---|
| **Aspect ratio** | Product-determined (bottle / box / sleeve / pouch) |
| **Glance time** | 1–2 seconds (shelf scan) |
| **Dwell time** | If picked up: 10–30 seconds (back-of-pack read) |
| **Sound default** | None |
| **Motion default** | Static; some product packaging uses AR triggers / dynamic codes |
| **Viewing context** | Retail aisle; at-home post-purchase |
| **Word-count ceiling** | Front-of-pack: 3–8 words primary; back-of-pack: 50–150 words |
| **Typical use cases** | Last-mile conversion at point-of-sale; brand expression through tactile / visual design; legal compliance (ingredients / warnings) |
| **Pitfalls** | Shelf competition fierce — primary readable in 1.5 sec or it loses. Legal requirements crowd back-of-pack. Print production complexity (Pantone matching, substrate decisions). |

## Owned web

Landing pages, homepage, product pages, pricing, blog, About, FAQ.

| Field | Value |
|---|---|
| **Aspect ratio** | Responsive — mobile portrait primary, desktop landscape secondary; design mobile-first |
| **Glance time** | 3–8 seconds for hero comprehension; 15s for above-the-fold decision-to-scroll |
| **Dwell time** | 30s–5+ min if engaged |
| **Sound default** | OFF — never autoplay sound; click-to-play required |
| **Motion default** | Variable — hero animations OK if respect prefers-reduced-motion; scroll-triggered motion is increasingly normalised but slow |
| **Viewing context** | Mobile (majority) — thumb-scroll; desktop (minority) — mouse + keyboard |
| **Word-count ceiling** | Hero: 8–14 words headline + 15–30 words subhead; full landing page: 300–1,500 words; longform / pillar page: 2,000–5,000 words |
| **Typical use cases** | Conversion (sign-up / buy / demo / download); SEO-target rank; product education; trust-building via case studies; AEO content for LLM citation |
| **Pitfalls** | Page-speed kills conversion (>3s load = 50% bounce). Mobile responsiveness untested. Cookie banners eating fold space. JavaScript-rendered content invisible to AEO. |

## Email

Marketing emails, transactional emails, newsletters.

| Field | Value |
|---|---|
| **Aspect ratio** | Width: 600–640px constrained; height: flexible |
| **Glance time** | Inbox-list view: 1–2 sec (subject + preview decision); opened email: 3–5 sec (hook decision) |
| **Dwell time** | 30–90 seconds if engaged; longer for newsletters with multiple items |
| **Sound default** | None (some clients block embedded audio; most users wouldn't expect it) |
| **Motion default** | Static; GIFs supported in most clients (animated header is common) |
| **Viewing context** | ~60% mobile, ~30% desktop, ~10% tablet; some users with images blocked by default; some in dark mode |
| **Word-count ceiling** | Subject: ≤50 chars (mobile preview cutoff ~33–41); preview: 40–90 chars; body: 90-second read = ~150–300 words for marketing; newsletters can be 800+ if structured |
| **Typical use cases** | Re-engaging opted-in audience; sequence-based nurture; transactional confirmation; newsletter content; cart abandonment |
| **Pitfalls** | Spam filters (subject triggers, image:text ratio, all-caps). Images blocked by default in many clients (Outlook). Dark mode rendering inconsistencies. Mobile preview cutoff. |

## Social — feed posts (static)

Single-image posts on LinkedIn / Instagram / Facebook / Threads.

| Field | Value |
|---|---|
| **Aspect ratio** | 1:1 (square, IG default) / 4:5 (portrait, IG max for feed) / 1.91:1 (LinkedIn, FB landscape link preview) |
| **Glance time** | 1.5–3 seconds before scroll-past |
| **Dwell time** | If hooked: 10–60 seconds (read caption + look at image) |
| **Sound default** | n/a (static) |
| **Motion default** | Static (definition) |
| **Viewing context** | Mobile feed scroll, thumb-navigation |
| **Word-count ceiling** | See [platform-native-social.md](platform-native-social.md) per platform; caption length varies LinkedIn 1,200–1,800 / IG hook in first 125 / FB 40–80 words |
| **Typical use cases** | Brand voice expression; visual storytelling; quote graphics; data viz; announcement |
| **Pitfalls** | Hero image weak = post invisible. Caption without hook in first line = no engagement. Same image across platforms ignoring aspect ratio = cropped or stretched. |

## Social — carousels

Multi-slide swipeable decks on Instagram, LinkedIn (PDF document upload), TikTok carousel.

| Field | Value |
|---|---|
| **Aspect ratio** | 1:1 or 4:5 (IG); 1.91:1 horizontal or 1:1 (LinkedIn PDF) |
| **Glance time** | Cover slide gets 1.5–3 sec to earn the first swipe |
| **Dwell time** | If swipe-engaged: 20–90 seconds across slides (~5–10 sec per slide average) |
| **Sound default** | n/a |
| **Motion default** | Static per slide; viewer-paced (swipe-driven) |
| **Viewing context** | Mobile feed, paused scroll, thumb-swipe |
| **Word-count ceiling** | Per slide: 3–8 word headline + 1–3 short lines body; 7–10 slides total |
| **Typical use cases** | Educational content (frameworks / how-tos); data visualisation series; storytelling-by-slide; product walkthrough; before-after narratives |
| **Pitfalls** | Cover slide failing = no swipes = no engagement. Too many slides = swipe-off. Slides without payoff = no save. Inconsistent visual register slide-to-slide = looks AI-coded. |

## Social — short-form video

TikTok, Instagram Reels, YouTube Shorts.

| Field | Value |
|---|---|
| **Aspect ratio** | 9:16 (vertical) always |
| **Glance time** | 1.5–3 seconds (the hook moment) |
| **Dwell time** | If hooked: 15–90 seconds (completion rate is the metric) |
| **Sound default** | OFF for ~50% of feed views — design sound-off-friendly with on-screen text carrying the hook |
| **Motion default** | Motion always; pattern-interrupt visual + on-screen text + spoken hook all firing in first 3 sec |
| **Viewing context** | Mobile, vertical, thumb-scroll, often muted in public spaces |
| **Word-count ceiling** | Script ~150 wpm spoken = 30s ≈ 70 words spoken + matching on-screen text in shorter bursts; caption ≤150 chars |
| **Typical use cases** | Hook-driven attention; trend participation; education in entertainment-coded format; product demos; behind-the-scenes; UGC-style brand content |
| **Pitfalls** | First 3 sec doesn't hook = scroll. Sound-required = half audience misses it. Trend used too late = looks corporate. Too-long content = drop-off mid-video. Production polish that signals "this is an ad" = low engagement. |

---

## Social — long-form video

YouTube long-form (>3 min), LinkedIn native video, Facebook video.

| Field | Value |
|---|---|
| **Aspect ratio** | 16:9 primary; 1:1 or 4:5 for in-feed autoplay social |
| **Glance time** | 15–30 sec to decide to keep watching (YouTube); 3 sec for autoplay feed (LinkedIn / FB) |
| **Dwell time** | YouTube: 3–15 min typical; can extend to 30–60+ for premium content; LinkedIn: 1–3 min |
| **Sound default** | ON for YouTube (audience clicks to play); OFF for in-feed social (autoplay muted) |
| **Motion default** | Motion |
| **Viewing context** | YouTube: desktop / mobile / TV / lean-back; LinkedIn: feed scroll, work-context |
| **Word-count ceiling** | YouTube 10-min video script ≈ 1,500 words at 150 wpm; LinkedIn 2-min ≈ 300 words |
| **Typical use cases** | Educational / explainer; brand storytelling; founder POV / vlog; product demos; thought leadership; documentary-style brand content |
| **Pitfalls** | First 30 sec didn't hook = drop. Production quality expectations high. Lacking captions = accessibility fail + lost sound-off viewers. Length without payoff. |

## Social — stories

Instagram Stories, Facebook Stories, LinkedIn Stories (deprecated but check), Snapchat.

| Field | Value |
|---|---|
| **Aspect ratio** | 9:16 vertical |
| **Glance time** | 1–3 sec per story slide before next-tap |
| **Dwell time** | 5–15 sec per slide if engaged; series of 5–10 slides typical |
| **Sound default** | OFF feed-default; ON if user taps |
| **Motion default** | Often video (short clips) or animated images |
| **Viewing context** | Mobile vertical, ephemeral (24h), tap-to-skip |
| **Word-count ceiling** | Per slide: 5–15 words on-screen text |
| **Typical use cases** | Behind-the-scenes; quick announcements; poll / question stickers; link-out via swipe-up; ephemeral content not designed for permanence |
| **Pitfalls** | Hook in first 1 sec or skip. Disposable feel can backfire for high-craft brands. Series too long = drop-off. Critical content shouldn't live ONLY in stories (gone in 24h). |

## Search ads

Google Ads, Bing Ads — paid search.

| Field | Value |
|---|---|
| **Aspect ratio** | n/a (text format primarily; image extensions exist) |
| **Glance time** | Scan of SERP — 1–2 sec per ad in the stack |
| **Dwell time** | Click decision is fast; landing page experience determines dwell |
| **Sound default** | None |
| **Motion default** | None |
| **Viewing context** | Search intent — user is actively looking |
| **Word-count ceiling** | Google: Responsive Search Ads — 15 headlines (30 chars each) + 4 descriptions (90 chars each); display URL + final URL |
| **Typical use cases** | High-intent demand capture; bottom-funnel; competitor-bidding; brand defence; service-area targeting |
| **Pitfalls** | Headline includes the keyword or it deprioritises. Generic ad copy loses to specific. Landing page mismatch with ad copy = quality score tanks. Ignoring extensions = lower CTR. |

## Display ads

Banner ads, programmatic display, retargeting placements.

| Field | Value |
|---|---|
| **Aspect ratio** | Many standard IAB sizes: 728×90 leaderboard, 300×250 medium rectangle, 320×50 mobile banner, 160×600 skyscraper, 970×250 billboard, 300×600 half-page |
| **Glance time** | 0.5–1.5 seconds (banner blindness is real) |
| **Dwell time** | n/a — click or skip |
| **Sound default** | OFF — display ads with sound get blocked in most placements |
| **Motion default** | Static (banners) or light motion (HTML5 with frame animation); video ads in larger placements |
| **Viewing context** | Adjacent to content; peripheral attention; often perceived as visual noise |
| **Word-count ceiling** | Headline ≤6 words; tag ≤10 words; CTA ≤3 words |
| **Typical use cases** | Retargeting (visited site, didn't convert); brand awareness (programmatic at scale); category targeting; sequential storytelling across placements |
| **Pitfalls** | Banner blindness — most display ads ignored. Creative ignoring placement size = looks broken. Animation that loops aggressively = annoying / ad-blocked. CTA buried. |

## Podcast / audio ads

Host-read ads, programmatic audio (Spotify / Acast / Apple Podcasts).

| Field | Value |
|---|---|
| **Aspect ratio** | n/a — audio |
| **Glance time** | 5–15 seconds before listener decides to skip ahead |
| **Dwell time** | Standard reads: 30 / 60 / 90 sec |
| **Sound default** | ON (audio context) |
| **Motion default** | n/a — audio |
| **Viewing context** | Commute / workout / chore / focused listening; often headphones; can't see the ad |
| **Word-count ceiling** | 60s read ≈ 150 words spoken |
| **Typical use cases** | Host-read endorsement (high-trust, high-cost); programmatic audio (lower cost, lower trust); brand awareness in podcasting-loyal audiences; DTC products targeting mass-market |
| **Pitfalls** | URL impossible to remember if complicated — use vanity URL or coupon code. Host-read scripts that don't fit the host's voice = backfires. Programmatic spots playing in wrong context. Boring spots get skipped. |

## Presentation / talk / keynote

TEDx, conference keynote, webinar, town hall, internal all-hands, scripted manifesto VO.

| Field | Value |
|---|---|
| **Aspect ratio** | Slides: 16:9 (standard) or 4:3 (legacy); 1:1 for some product / creative talks |
| **Glance time** | First 30–60 sec of a talk decides whether audience leans in or checks email |
| **Dwell time** | Full slot — TEDx 12–18 min; keynote 30–45 min; webinar 30–60 min; town hall 15–30 min |
| **Sound default** | ON (speaker amplified) |
| **Motion default** | Slides typically static with light transitions; sometimes embedded video |
| **Viewing context** | Live audience (in-person) or remote (Zoom / streaming); attention competing with phone / second screen |
| **Word-count ceiling** | Spoken at ~150 wpm → 15 min talk ≈ 2,250 words; manifesto VO slower at ~120 wpm |
| **Typical use cases** | Thought leadership; product launch keynote; brand storytelling; internal alignment; teaching / workshop |
| **Pitfalls** | Slides crowded with text = audience reads instead of listens. No memorable line = nothing to tweet. Reading from script = audience disconnects. Speaker voice unrehearsed = stutter / pace problems. |

## Live event / experiential

Pop-ups, brand activations, conference booths, immersive experiences.

| Field | Value |
|---|---|
| **Aspect ratio** | Physical space — varied |
| **Glance time** | Passerby: 2–5 sec; engaged visitor: 1–10 min |
| **Dwell time** | If activated: 5–30 min in-experience |
| **Sound default** | Variable — ambient (background music), interactive (touch-triggered), or none |
| **Motion default** | Often motion (screens, video walls, kinetic elements); some pure static (installations) |
| **Viewing context** | Physical, embodied, social (often with friends / colleagues); shareable for social |
| **Word-count ceiling** | Sign / wall: 10–20 words; explainer panels: 50–100; printed handout: variable |
| **Typical use cases** | Brand activation at events; PR moments; product trial; community building; UGC generation (designed-for-Instagram moments) |
| **Pitfalls** | Cost-per-touch high. Hard to measure. Requires production logistics. UGC potential underused. Inaccessible (physical disability, geographic restriction). |

## Channel-to-form-to-agent quick reference

| Channel | Primary Copywriter | Primary Designer |
|---|---|---|
| OOH | Short-form (headline) | Static |
| Broadcast video | Presentation (script) or Long-form (VO narration) | Motion |
| Cinema | Same as broadcast | Motion |
| Print | Short-form (headline) + Long-form (body) | Static |
| Direct mail | Short-form (envelope teaser) + Email-style (letter body) | Static |
| Packaging | Short-form (front) + Long-form (back) | Static |
| Owned web | Web | Static + Motion (hero motion) |
| Email | Email | Static (graphics) + Motion (animated headers) |
| Social feed (static) | Social/Community Manager | Static |
| Social carousels | Social/Community Manager | Static |
| Social short-form video | Social/Community Manager (script) | Motion |
| Social long-form video | Long-form (narrated script) or Presentation (interview) | Motion |
| Social stories | Social/Community Manager | Static + Motion |
| Search ads | Short-form | n/a (mostly text) |
| Display ads | Short-form | Static (banner) or Motion (HTML5) |
| Podcast ads | Short-form (script) | n/a (audio) |
| Presentation / talk | Presentation | Static (slides) + occasional Motion (embedded clips) |
| Live event | Various — Short-form (signage) + Long-form (panel walls) | Static + Motion (installations) |
